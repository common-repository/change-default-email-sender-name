== Description ==

Change Default Email Sender Name is a simple plugin that allows you to change the sender name and Email in your WordPress Website's outgoing emails.

To replace the default WordPress settings, go to Settings / Change Sender Name Setting and add Sender Detail.

No coding knowledge is required.

Simply download and install the Change Default Email Sender Name plugin. 


== Installation ==
1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. change will be automatically seen in email sender name

== Frequently Asked Questions ==

 How can I use this plugin?
 You can download it and active it. go to Settings / Change Sender Detail Setting and add Sender Name and Email.
